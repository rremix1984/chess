## 游戏演示

### 启动方式

1. **通过游戏中心**：
   ```bash
   ./start-game.sh
   ```
   然后点击"💰 大富翁"按钮

2. **直接启动**：
   ```bash
   ./start-monopoly.sh
   ```

3. **Maven命令**：
   ```bash
   mvn exec:java -pl monopoly -Dexec.mainClass="com.example.monopoly.MonopolyFrame"
   ```

### 游戏流程

1. 启动游戏后看到棋盘界面
2. 点击"开始游戏"初始化4个玩家
3. 玩家轮流点击"掷骰子"移动
4. 落到地产格时选择是否购买
5. 游戏日志记录所有事件
6. 可随时返回主菜单
